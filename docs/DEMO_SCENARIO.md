# Scenario de demonstration soutenance (10-12 min)

## 1) Initialisation
1. Lancer la base et l'API.
2. Appliquer les migrations Alembic.
3. Executer le seed.

## 2) Parcours visiteur
1. Ouvrir la page d'accueil.
2. Consulter les publications validees.
3. Envoyer un message via la page contact.

## 3) Parcours membre
1. Creer un compte membre.
2. Se connecter.
3. Lier ORCID et importer les publications.
4. Uploader un document scientifique.
5. Exporter son bilan PDF membre.

## 4) Parcours administrateur
1. Se connecter en admin.
2. Consulter le dashboard admin.
3. Ouvrir la file de moderation.
4. Valider une publication/communication.
5. Consulter l'historique des validations.
6. Exporter le bilan global labo (PDF).

## 5) Cloture
1. Montrer la structure technique (Next.js, FastAPI, PostgreSQL).
2. Montrer les tests automatises backend.
3. Conclure sur les evolutions futures (V2).
